Cupid Toolbox

Version of 2019-05-19

Cupid is a MATLAB toolbox for working with probability distributions (e.g., for data analysis or simulation).

  • For a quick look at what it does, look at this starter demo: DemoCupid1.m.
  • Here is the complete documentation as a PDF file.

Other demos

  • Parameter estimation
  • Confidence intervals for parameter estimates
  • Multivariate random number generation
  • Constructing a complicated distribution(e.g.,log-Pearson III)
  • Probit analysis
  • Using MATLAB distribution objects
